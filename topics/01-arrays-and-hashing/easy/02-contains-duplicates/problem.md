# Contains Duplicate

**Difficulty:** Easy
**Topic:** Hashes
**Language:** Ruby
**LeetCode:** [#217](https://leetcode.com/problems/contains-duplicate/)
**Interview Frequency:** Very High

---

## Problem Statement

Given an array of integers `nums`, return `true` if any value appears **at least twice**.
Return `false` if every element is distinct.

---

## Examples

```
Input:  [1, 2, 3, 1]
Output: true

Input:  [1, 2, 3, 4]
Output: false

Input:  [1, 1, 1, 3, 3, 4, 3, 2, 4, 2]
Output: true
```

---

## Constraints

- Array can be empty → return `false`
- Array can contain negative numbers
- Array can have a single element → return `false`
